# NSL-KDD IDS Research Console

Static final-year research showcase for an NSL-KDD intrusion detection project.

The app includes:

- Multipage project overview, dataset notes, methodology, and results.
- A browser-side tester for the exported final XGBoost model.
- Saved evaluation plots from the research artifacts.
- No paid hosting, database, or inference API dependency.

The live model tester loads `assets/data/xgb_final.json` plus `assets/data/model_metadata.json` and runs XGBoost tree traversal in JavaScript.
